## NTSC
NTSC信号は
白黒テレビとの後方互換性を維持しつつ
半世紀以上前のアナログ技術で、ひとつの信号ラインに水平同期信号、垂直同期信号、カラー信号を載せるという
先人達の知識と知恵と工夫が盛り込まれた素晴らしい規格だと思います。
